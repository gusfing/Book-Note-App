## 🎯 Overview

Book-Note is a lightweight web application for book lovers. Keep track of:
- 📖 **Books you've read** — Add title, author, genre
- 📝 **Personal notes** — Write short reflections on each book
- ⭐ **Ratings** — Rate books 1-5 stars
- ❤️ **Favorites** — Mark books as favorites
- 🔍 **Search** — Find books quickly

## ✨ Features

- ➕ **Add Books** — Title, author, genre, cover image URL
- 📝 **Take Notes** — Write thoughts on each book
- ⭐ **Star Rating** — 1-5 star rating system
- ❤️ **Favorite Marking** — Quick-access to best books
- 📊 **Reading Stats** — Books read, average rating, genres breakdown
- 🎨 **Clean UI** — Minimal, book-focused design

## 🚀 Getting Started

```bash
# Clone
git clone https://github.com/gusfing/Book-Note-App.git
cd Book-Note-App

# Open in browser
open index.html

# Or use local server
npx serve .
```
